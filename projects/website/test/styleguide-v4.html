01 · Type

Two voices, strict roles.

v4 is a two-voice typographic system. Bricolage Grotesque carries display work — hero titles, section headers, prices, stats. Beatrice carries body, UI, and the warm italic accents that used to be Orpheus. There is no third typeface.

03 · Buttons

Pills, always.

SPS buttons are pills (border-radius 999px). This is the strongest brand override on top of shadcn's default rounded-rect Button. Pills are kept even at small sizes — the shape is the brand signature across dashboard, homepage, and product flows.

08 · Direct response

Design for the job, not the catalog.

SPS is an ecommerce business. Landing pages and product creation flows benefit from direct-response thinking. These principles are guidance, not handcuffs — fold them in where they sharpen the work. They apply to marketing surfaces, not dashboards or admin tools.

— 01

Think in jobs, not products

Customers don't want "framed prints" — they want to frame a meaningful photo. Headlines describe what the customer accomplishes, not what the product is.

— 02

Lead with action

Every page has one thing it wants the user to do. CTAs start with verbs: "Make a book", "Frame a photo", "Send a gift". Never "Shop now" or "Learn more".

— 03

Guide, don't catalog

Instead of a wall of options, walk the user through a flow. One decision at a time. Use the stepper component for multi-step builders.

— 04

Maintain the scent

If a user clicks an ad about "the perfect gift for grandma", the landing page should echo it visually, verbally, tonally. Build contextual landing pages, not generic ones.

— 05

Customer is the hero

Position the customer as the protagonist. SPS is the guide. Empathy + authority. Not "look at our great products" — "you have a story, here's how to keep it."

— 06

Benefit-driven copy

Lead with what the customer gains, not what we built. "Memories that last forever" before "archival ink." Use the "so what?" test.

Don't write

  • Shop our products
  • Learn more about photo books
  • Unlock unparalleled photo experiences
  • Click here for our innovative ecosystem

Do write

  • Make a book
  • See how a layflat album gets made
  • Photos you'd actually keep
  • Start her gift in 90 seconds
When this applies · when it doesn't

DR thinking applies to marketing surfaces: landing pages, product detail pages, campaign entry points, creation flows. It does not apply to internal dashboards, admin tools, developer documentation, or pure informational pages — those should be clear, dense, and direct rather than persuasive.

10 · Building real shadcn

Drop the tokens, get the brand.

When scaffolding a real Next.js + shadcn project for SPS, the entire brand transfer is one CSS variable block plus one component override. Here's the recipe.

The 8-step shadcn setup for SPS v4

  1. Scaffold: npx create-next-app@latest sps-project --typescript --tailwind --app
  2. Init shadcn: cd sps-project && npx shadcn@latest init
  3. Add components: npx shadcn@latest add button card table badge input switch alert dialog dropdown-menu tabs sheet select radio-group separator avatar
  4. Theme: Open app/globals.css and paste the entire :root block from §2 of this styleguide (or styleguide-v4.md §3.2) into the file. shadcn's default :root can be replaced wholesale.
  5. Fonts: Add Bricolage Grotesque via next/font/google and Beatrice via next/font/local using the woff2 files (or reference the S3 URLs from §11 of this guide).
  6. Tailwind config: In tailwind.config.ts, add fontFamily.sans = ['var(--font-beatrice)', ...] and fontFamily.display = ['var(--font-bricolage)', ...].
  7. Pill button override: Open components/ui/button.tsx, find the Button's cva() base classes, and change rounded-md to rounded-full. This is the only meaningful brand override on top of shadcn defaults.
  8. Build: Use the page archetypes in §12 of styleguide-v4.md as references. The dashboard, homepage, photo flow, and PDP are all built from these primitives.

That's the entire brand transfer. The CSS variable block does 95% of the work. The Button override does the remaining 5%.
